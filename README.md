# Vuln-Scan
A free, open-source tool to detect security vulnerabilities in:
- Source code (JS, Python, Java, etc.)
- Docker & Kubernetes configs
- Swagger/OpenAPI specs
- YAML, JSON, .env files
- Any text-based file

## Usage

1. Visit: https://yourusername.github.io/vulnscan/
2. Upload a file or paste code
3. Click "RUN SCAN"
4. View findings and fixes

## Features

- 50+ security rules
- Works on Windows, macOS, Linux
- No installation required
- No data sent to servers (100% local)
- Step-by-step fix guidance
